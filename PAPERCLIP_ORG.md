# Paperclip Org Design (RelayRank)

## Company mission
Build the highest-leverage organic growth engine for early-stage B2B SaaS.

## Org chart (initial)
- CEO Agent (strategy + approvals)
- COO Agent (workflow orchestration)
- SEO Strategist Agent
- Content Lead Agent
- Writer Agent A
- Writer Agent B
- Distribution Agent
- QA/Editor Agent

## Heartbeats
- CEO: every 12h (strategy + KPI review)
- COO: every 4h (ticket triage, blocker clearing)
- SEO Strategist: every 24h (keyword refresh)
- Content Lead: every 8h (brief queue management)
- Writers: every 6h (draft progress)
- Distribution: every 12h (post scheduling)
- QA/Editor: every 6h (quality gate)

## Governance defaults
- Human approval required for:
  - publishing content
  - sending outbound email/DM
  - changing pricing
- Monthly budget cap per agent set before activation

## Ticket lanes
1. Strategy
2. Production
3. Distribution
4. Reporting
5. Client Ops
